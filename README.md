# API To do Inference in Deep Learning Models

Libraries:
- Flask
- Tensorflow 2.x
- Numpy
- openCv

## How to run:

To start api:


```
python app.py --address '0.0.0.0' --port 5000 --model_path './model.h5'
```

That's it! It's serving a saved Keras model to you via Flask.

To use the service:

run `test.ipynb`
